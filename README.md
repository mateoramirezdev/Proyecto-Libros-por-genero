# Proyecto: Catálogo de Librería – C++

Sistema en C++ que permite cargar, organizar y listar libros por género utilizando listas basadas en arrays.

## Funcionalidades
- Carga de libros (título, autor, género)
- Listados por género
- Reportes básicos de cantidad de libros por categoría, entre otros.

## Tecnologías usadas
- Listas basadas en arrays
- Modularización
- Estructuras simples y validaciones
- Archivos binarios

## Objetivo
Practicar organización de datos y estructura de listas dentro de un sistema de registro realista.
